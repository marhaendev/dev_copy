# dev_copy
copy html using class
## Requirement
- jquery
- style.css
- script.css

### code
```
<script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>
```

## Using
- make container and add **dev**
- use class **_copy**

## example
```
<div class="dev">
  <div class="copy"> text example</div>
</div>
```
### result
```
<div class="copy"> text example</div>
```
